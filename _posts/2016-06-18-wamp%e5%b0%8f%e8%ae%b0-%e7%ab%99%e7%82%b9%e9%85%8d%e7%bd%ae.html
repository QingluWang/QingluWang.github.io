在windows上搭建一个服务器环境，一般用WAMPSERVER即可，测试在windows上写好的php网页也有多种方法
一、仅改动apache默认站点路径
在【wamp路径】\bin\apache\apachex.x.x\conf里找到apache配置文件--httpd.conf,寻找documentroot
DocumentRoot "改为需要运行的网站路径"

#
# Each directory to which Apache has access can be configured with respect
# to which services and features are allowed and/or disabled in that
# directory (and its subdirectories).
#
# First, we configure the "default" to be a very restrictive set of
# features.
#

AllowOverride none
Require all denied

#
# Note that from this point forward you must specifically allow
# particular features to be enabled - so if something's not working as
# you might expect, make sure that you have specifically enabled it
# below.

PS:不要在网站路径里出现中文！
二、配置单个站点或多个站点
单个站点
1.在【wamp路径】\wamp\bin\apache\apachex.x.x\conf\extra里找到httpd-vhosts.conf添加

DocumentRoot "改为需要运行的网站路径"
ServerName 主机名，在网址栏中输入可以直接访问

2.在【wamp路径】\bin\apache\apachex.x.x\conf里找到apache配置文件--httpd.conf,寻找onlineoffline，改为如下
# onlineoffline tag - don't remove
Order Deny,Allow
Allow from all
#允许从外部访问
Require all granted

3.在【wamp路径】\bin\apache\apachex.x.x\conf里找到apache配置文件--httpd.conf,寻找httpd-vhosts.conf，引用该文件
# Virtual hosts
#Include conf/extra/httpd-vhosts.conf
改为
# Virtual hosts
Include conf/extra/httpd-vhosts.conf
4.在C:\Windows\System32\drivers\etc中找到HOSTS文件，编辑
最后面添加
127.0.0.1 想要在网址栏中直接访问的主机名
多个站点
跟单个站点差不多，只是第一步和最后一步里添加相应站点信息
PS:可能会出现You don't have permission to access / on this server
为此我的解决方法是，把localhost作为一个"站点"添加。。
